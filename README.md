## 项目简介

本项目是浙江树人学院信息科技学院《人工智能》课程大作业，基于 Web 技术开发的台州府城智慧文旅导览系统。系统以"智慧文旅"为主题，融合了 AI 大模型智能问答、智能行程规划、当季打卡推荐、地图交互漫游、实时天气查询等核心功能，为游客提供沉浸式、个性化的旅游体验。

**设计理念**：系统以台州府城（临海）的历史文化底蕴为背景，结合 AIGC 技术生成图文资产，通过直观的地图交互操作和 AI 对话能力，帮助游客快速了解景区景点、规划游览行程、获取实时天气资讯、聆听景点语音讲解，实现"一屏览尽千年府城"的智慧旅游愿景。

**适用场景**：景区游客服务中心、文旅宣传展示、智慧城市文旅示范项目等。

## 功能介绍

### 🗺️ 核心功能一：景点地图导览 + VR全景漫游

- **交互式地图**：基于 HTML img + CSS Transform 实现，支持拖拽漫游、滚轮缩放、自动漂移漫游
- **四季切换**：春、夏、秋、冬四种主题地图自动切换
- **景点标记**：彩色光晕圆点标记各景点位置，点击查看详情
- **区域快速跳转**：支持紫阳古街、台州府城墙、东湖、巾山等区域列表快速聚焦
- **VR全景**：嵌入720云全景链接，支持360度沉浸式浏览
- **高德导航**：一键跳转高德地图URI实现景点导航

### 🤖 核心功能二：智能AI问答（智慧导览）

- **多API降级策略**：集成6个AI服务（DeepSeek、xyhelper、ChatAnywhere、Moonshot、OpenRouter、Pollinations），依次尝试降级
- **智能问答**：支持景点介绍、历史文化问答、游玩建议、美食推荐、住宿交通咨询
- **语音播报**：Web Speech API 实现文字转语音（TTS），默认播放点击静音
- **历史时间线**：台州府城从唐代建城至今的历史大事记纵向展示
- **知识长廊**：卡片轮播展示文化科普内容

### 📅 核心功能三：智能行程规划

- **官方推荐路线**：经典环线/文化深度两条路线，时间线卡片展示
- **智能推荐**：基于兴趣类型选择的个性化行程方案生成
- **预算估算**：门票/餐饮/交通/住宿费用计算展示
- **当季打卡**：根据月份自动识别季节，推荐5个必打卡景点
- **视频推荐**：聚合当季精彩视频内容
- **新闻资讯**：最新文旅资讯聚合展示
- **天气预报**：7天天气预报查询

### 🏠 首页欢迎展示

- **视频背景**：全屏台州府城宣传视频，循环静音播放
- **天气查询**：实时温度和天气状况显示（腾讯天气 + Open-Meteo 并行请求）
- **季节装饰**：根据季节显示飘落花瓣/雨滴/雪花等动画
- **品牌展示**："台州府城旅游区"大字逐字出现动画
- **全局导航**：底部4个导航按钮切换页面

## 技术栈

| 分类 | 技术 |
|------|------|
| 前端技术 | 原生 HTML5 + CSS3 Flex/Grid布局 + 毛玻璃效果 + CSS变量主题切换 |
| 交互引擎 | 纯 JavaScript ES6+，模块化函数设计，事件驱动架构 |
| 地图系统 | HTML img + CSS Transform，支持拖拽、缩放、自动漫游、坐标聚焦 |
| AI集成 | RESTful API调用（DeepSeek、Moonshot等6个云端大模型） |

| 导航服务 | 高德地图 URI Scheme |
